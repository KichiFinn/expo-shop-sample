# expo-shop-sample
A sample shop application using [react native](https://reactnative.dev/) and [expo](https://expo.io/).

# Getting Started
1. Install dependency
   ```console
   $ npm install
   ```
2. Android
   ```console
   $ npm run android
   ```
3. iOS
   ```console
   $ npm run ios
   ```
